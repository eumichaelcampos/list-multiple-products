== Description ==
Um plugin que permite listar e selecionar múltiplos produtos em uma página com shortcode, com uma interface de administração para configurar produtos padrão.

== Installation ==
1. Faça o upload dos arquivos do plugin para o diretório `/wp-content/plugins/list-multiple-products` ou instale diretamente pelo repositório de plugins do WordPress.
2. Ative o plugin através do menu 'Plugins' no WordPress.
3. Configure os produtos padrão a serem exibidos através da página de administração "List Multiple Products".
4. Use o shortcode `[list-product]` para listar os produtos configurados na administração ou `[list-product products-id="123,456,789"]` para listar produtos específicos.
5. Para listar todos os produtos, habilite a opção "List All Products" na página de administração e configure o número de produtos por página.

== Frequently Asked Questions ==
= Como uso o shortcode? =
Use o shortcode `[list-product]` para listar os produtos configurados na administração ou `[list-product products-id="123,456,789"]` para listar produtos específicos. Para listar todos os produtos, habilite a opção "List All Products" na página de administração.
